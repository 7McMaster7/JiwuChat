## Introducción

JiwuChat es una aplicación de chat multiplataforma ligera `(~10MB)` construida con Tauri2 y Nuxt3. Ofrece una variedad de funciones de mensajería en tiempo real, bots de chat grupales de IA (con `iFlytek Spark`, `KimiAI`, etc. ya integrados), `llamadas de audio/video WebRTC`, pantalla compartida y funciones de compra con IA. Admite una comunicación fluida entre dispositivos, cubriendo varios tipos de mensajes como texto, imágenes, archivos y voz. También admite chats grupales y configuraciones personalizables. Ofrece modos claro/oscuro para potenciar una red social eficiente. ✨
