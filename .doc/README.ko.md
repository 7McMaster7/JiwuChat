## 소개

JiwuChat은 Tauri2와 Nuxt3로 구축된 경량`(~10MB)` 크로스 플랫폼 채팅 애플리케이션입니다. 실시간 메시징 기능, AI 그룹 채팅 봇(`iFlytek Spark`, `KimiAI` 등 이미 통합됨), `WebRTC 오디오/비디오 통화`, 화면 공유 및 AI 쇼핑 기능을 포함한 다양한 기능을 제공합니다. 텍스트, 이미지, 파일, 음성 등 다양한 메시지 유형을 포괄하는 장치 간 원활한 통신을 지원합니다. 또한 그룹 채팅 및 사용자 정의 가능한 설정을 지원합니다. 효율적인 소셜 네트워크를 강화하기 위해 라이트/다크 모드를 제공합니다. ✨

## ⏳ 시작하기

### 📦 종속성 설치

```sh
# node version > = 20
npm install -g pnpm

pnpm install
```
