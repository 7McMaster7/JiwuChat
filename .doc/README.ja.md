## 紹介

JiwuChatは、Tauri2とNuxt3で構築された軽量`（〜8MB）`マルチプラットフォームチャットアプリケーションです。様々なリアルタイムメッセージング機能、AIグループチャットボット（`iFlytek Spark`、`KimiAI`など統合）、`WebRTC音声/ビデオ通話`、画面共有、AI搭載ショッピングをサポートしています。デバイス間でのシームレスな通信を可能にし、テキスト、画像、ファイル、音声メッセージ、グループチャット、カスタマイズ可能な設定をサポートしています。効率的なソーシャルネットワーキングのためのライト/ダークモードが利用可能です。✨

## ⏳ はじめに

### 📦 依存関係のインストール

```sh
Node.js >= 18 が必要
npm install -g pnpm

pnpm install
```

### ✨ 開発

- 📌 バックエンドサーバーが**ない**場合、`.env.development`を変更するか`.env.production`を使用してください。

```sh
ターミナル1：Nuxtを開始（本番）
pnpm run prod:nuxt
ターミナル2：Tauriを開始
pnpm run dev:tauri
```

- バックエンドが**ある**場合、開発用に`.env.development`をカスタマイズしてください：

```sh
別々に実行することを推奨
ターミナル1：Nuxtを開始
pnpm run dev:nuxt
ターミナル2：Tauriを開始
pnpm run dev:tauri
```

### 📦 ビルド

```sh
pnpm run build:tauri
```

### ❌ pnpmインストールエラー

レジストリを確認：

```sh
pnpm get registry
```

一時的な修正：

```sh
pnpm --registry https://registry.npm.taobao.org install any-touch
```

永続的な変更：

```sh
pnpm config set registry https://registry.npm.taobao.org
```

元に戻す：

```sh
pnpm config set registry https://registry.npmjs.org
```
