### 介绍

极物聊天 Tauri APP 是一个基于Nuxt3 + Tauri + element-plus 基本框架的聊天桌面应用。它提供了一个美观的用户界面，并集成了聊天和社交功能。同时，它还支持AI购物聊天和全局夜间模式。用户可以通过该应用进行实时聊天、分享动态和AI客服问答。

## ⏳ 起步 | Project Setup

### 📦 安装依赖

```sh
# node 版本 >= 16
npm install -g pnpm

pnpm install
```

### ✨ 开发

- 若有后端服务，可采用自定义修改开发`.env.development`环境变量文件进行开发

```sh
# 建议分开运行
# 终端1：启动nuxt
pnpm run dev:nuxt
# 终端2：启动tauri
pnpm run dev:tauri
```
- 若没有后端服务，修改`.env.development`环境变量，或使用`.env.production`配置文件。

```sh
# 终端1：启动nuxt
pnpm run dev:nuxt:prod
# 终端2：启动tauri
pnpm run dev:tauri
```

### 📦 打包

```sh
pnpm run build:tauri
```

### ❌ pnpm install error

查看源

```sh
pnpm get registry
```

临时修改

```sh
pnpm --registry https://registry.npm.taobao.org install any-touch
```

持久使用

```sh
pnpm config set registry https://registry.npm.taobao.org
```

还原

```sh
pnpm config set registry https://registry.npmjs.org
```
